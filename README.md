# shortcut  

F1: 打开undo tree  
F2: 生成tag  
F3: 为当前头文件生成防止重复包含的宏

ctrl + ]: 跳到定义处  
ctrl + o: 跳回

在buffer标签页上点v键: 将buffer垂直打开

gx : 打开超链接



16进制模式：

[doc][2]

ctrl + h: 开启 / 关闭  
r：修改当前光标下的值
